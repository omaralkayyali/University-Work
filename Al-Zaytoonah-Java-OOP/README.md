# Al-Zaytoonah-Java-OOP
# مساق البرمجة الكينونية - جامعة الزيتونة الأردنية

## Overview
This repository contains a fully organized Java OOP course project for Al-Zaytoonah University of Jordan. It includes official exam solutions, structured practice sheets, and complete source code for key object-oriented programming concepts.

يتضمن هذا المستودع مشروعًا كاملاً لمساق البرمجة الكينونية بجافا لجامعة الزيتونة الأردنية. يحتوي على حلول الامتحانات الرسمية، أوراق العمل المنظمة، وشيفرات برمجية كاملة لمفاهيم البرمجة الكينونية الأساسية.

## How to Run
1. Open the chosen folder in your IDE or terminal.
2. Compile the Java files:
```bash
javac *.java
```
3. Run the main class in the folder, for example:
```bash
java ElevatorSystem
```

---

## Notes
- Java version: `Java 8` or newer
- All code is provided as standalone, package-free classes for easy compilation and execution.

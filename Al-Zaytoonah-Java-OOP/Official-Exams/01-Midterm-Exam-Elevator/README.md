# Midterm Exam - Elevator
# امتحان منتصف الفصل - المصعد

## Overview
This folder contains the midterm exam solution for the Elevator class. The solution includes speed and floor validation, cost calculations, and a main program that manages an array of 5 elevators.

يحتوي هذا المجلد على حل امتحان منتصف الفصل لكلاس المصعد. يشمل الحل التحقق من السرعة والطوابق، حساب التكاليف، وبرنامج رئيسي يدير مصفوفة من 5 مصاعد.

## Requirements
- `speed` must be `'F'` or `'S'`
- `floors` must be between `1` and `20`
- `getSpeedCost()` returns `3000` for `F` and `1500` for `S`
- `getFloorCost()` returns tiered cost based on floor count
- `getTotalCost()` returns total cost plus `100`

# Final Exam Form A - Programming Contest
# الامتحان النهائي نموذج أ - نظام مسابقات البرمجة

## Overview
This folder contains the final exam solution for the contest management system. It includes the base Contest class, the Team class, and a ProgrammingContest subclass that manages teams via an ArrayList.

يحتوي هذا المجلد على حل الامتحان النهائي لنظام إدارة المسابقات. يشمل الكلاس الأساسي Contest، وكلاس الفريق Team، والفئة ProgrammingContest التي تدير الفرق بواسطة ArrayList.

## Requirements
- `Contest` validates `name`, `level`, and `location`
- `level` must be one of `A`, `B`, `C`
- `RegFee()` returns a fee based on the level and location
- `Team` validates `id`, `name`, and `members`
- `ProgrammingContest` supports `addTeam`, `removeTeam`, `searchTeamByName`, and team display

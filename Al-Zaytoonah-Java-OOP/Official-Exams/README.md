# Official Exams
# الامتحانات الرسمية

## Overview
This folder contains the official exam tasks for the Java OOP course. Each subfolder holds a complete, ready-to-run solution with clear validation rules and object-oriented design.

يحتوي هذا المجلد على مهام الامتحانات الرسمية لمساق البرمجة الكينونية في جافا. يحتوي كل مجلد فرعي على حل متكامل وجاهز للتشغيل مع قواعد تحقق واضحة وتصميم كائني.

## Sections
- `01-Midterm-Exam-Elevator/` - Elevator class with speed and floors validation.
- `02-Final-Exam-Form-A/` - Contest system with teams and ArrayList management.
- `03-Final-Exam-Incomplete/` - Cosmetics system with Perfume and Musk inheritance.

## About
Each section includes:
- Java source files with complete class definitions
- A main demo file for executing the scenario
- Clear validation in constructors and setters

كل قسم يتضمن:
- ملفات جافا مع تعريفات كاملة للكلاسات
- ملف رئيسي لتشغيل السيناريو
- تحقق واضح داخل الباني ودوال setter

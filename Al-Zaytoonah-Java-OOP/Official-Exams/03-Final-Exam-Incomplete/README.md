# Final Exam Incomplete - Cosmetics Store
# الامتحان النهائي غير المكتمل - متجر مستحضرات

## Overview
This folder contains the incomplete final exam task for a cosmetics store. The solution includes a base Perfume class, a Musk subclass, and a CosmeticsStore manager with searching and filtering.

يحتوي هذا المجلد على مهمة الامتحان النهائي غير المكتملة لمتجر مستحضرات. يشمل الحل كلاس العطر الأساسي Perfume، وكلاس الميسك Musk، ومدير المتجر CosmeticsStore مع البحث والتصفية.

## Requirements
- `Perfume` enforces `price` between `20` and `180`
- `Perfume` enforces valid `scent` values such as `O`, `F`, `W`, or `T`
- `Musk` adds `id` and `name`, where `name` must be `White` or `Black`
- `CosmeticsStore` holds `ArrayList<Musk>` and provides interactive search via Scanner

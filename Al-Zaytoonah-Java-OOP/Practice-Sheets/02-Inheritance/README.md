# Inheritance Practice
# تمرين الوراثة

## Overview
This folder contains inheritance exercises where subclasses extend base classes and override behavior for increased flexibility.

يحتوي هذا المجلد على تمارين الوراثة حيث تُمدد الفئات الفرعية الفئات الأساسية وتُعيد تعريف السلوك لزيادة المرونة.

## Exercises
- `Computer.java` and `Laptop.java` - laptop extends computer and overrides total price calculation.
- `WeatherData.java` and `Forecast.java` - forecast extends weather data and adds comfort index calculation.

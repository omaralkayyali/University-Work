# Composition & Aggregation Practice
# تمرين التكوين والتجميع

## Overview
This folder contains examples of composition and aggregation. The code demonstrates how objects can contain other objects either tightly (composition) or loosely (aggregation).

يحتوي هذا المجلد على أمثلة للتكوين والتجميع. يوضح الكود كيف يمكن للكائنات أن تحتوي على كائنات أخرى بثبات (التكوين) أو بمرونة (التجميع).

## Exercises
- `Composition/` - smartphone components using `Battery` and `Screen`.
- `Aggregation/` - a `Room` manages an `ArrayList<Furniture>`.
- `PersonName/` - composition example where `Person` owns a `Name` object.

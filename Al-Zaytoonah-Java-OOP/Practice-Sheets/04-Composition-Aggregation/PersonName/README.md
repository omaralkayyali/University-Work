# Person & Name Composition
# تكوين Person و Name

## Overview
This folder demonstrates a small composition example: a `Person` contains a `Name` object. The name is created and managed inside the person.

يعرض هذا المجلد مثالاً صغيراً للتكوين: `Person` يحتوي على كائن `Name`. يتم إنشاء الاسم وإدارته داخل الشخص.

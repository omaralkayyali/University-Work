# Composition Example
# مثال التكوين

## Overview
This folder demonstrates composition in Java: a `SmartPhone` is built from `Battery` and `Screen` objects. The smartphone cannot exist without its required components.

يُظهر هذا المجلد التكوين في جافا: يتم بناء `SmartPhone` من كائنات `Battery` و `Screen`. الهاتف لا يمكن أن يوجد بدون مكوناته الأساسية.

## Files
- `Battery.java`
- `Screen.java`
- `SmartPhone.java`
- `CompositionDemo.java`

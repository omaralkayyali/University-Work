# Aggregation Example
# مثال التجميع

## Overview
This folder demonstrates aggregation in Java: a `Room` manages a list of `Furniture` items, where the furniture exists independently from the room.

يُظهر هذا المجلد التجميع في جافا: `Room` يدير قائمة من عناصر `Furniture`، حيث يمكن أن توجد الأثاث بشكل مستقل عن الغرفة.

## Files
- `Furniture.java`
- `Room.java`
- `AggregationDemo.java`

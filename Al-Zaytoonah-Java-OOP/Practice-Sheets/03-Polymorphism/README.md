# Polymorphism Practice
# تمرين تعدد الأشكال

## Overview
This folder includes polymorphism exercises in which a parent class is used with different child classes to demonstrate dynamic method binding.

يحتوي هذا المجلد على تمارين تعدد الأشكال حيث يتم استخدام فئة الأب مع فئات الطفل المختلفة لإظهار الربط الديناميكي للدوال.

## Exercises
- `Food.java` - parent class with a nutritional summary method.
- `Fruit.java` and `Vegetable.java` - override the summary method.
- `PolymorphismDemo.java` - runs an ArrayList of `Food` objects and demonstrates polymorphism.

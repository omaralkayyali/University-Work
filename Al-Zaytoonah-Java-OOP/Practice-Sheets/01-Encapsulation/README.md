# Encapsulation Practice
# تمرين الكبسولة

## Overview
This folder contains encapsulation exercises that focus on data validation and protected access using private fields and setters/getters.

يحتوي هذا المجلد على تمارين الكبسولة التي تركز على التحقق من صحة البيانات والوصول المحمي باستخدام الحقول الخاصة وطرق setter/getter.

## Exercises
- `Smartphone.java` - validates brand and storage capacity.
- `TV.java` - validates channel, volume, and power state.
- `SmartphoneDemo.java` and `TVDemo.java` - run interactive examples.

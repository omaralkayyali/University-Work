# Practice Sheets
# أوراق العمل والتمارين التطبيقية

## Overview
This folder includes structured practice exercises for the Java OOP course. Exercises are grouped by concept so students can learn progressively from encapsulation to advanced composition and aggregation.

يحتوي هذا المجلد على تمارين منظمة لمساق البرمجة الكينونية بجافا. تم تجميع التمارين حسب المفهوم ليتمكن الطالب من التعلم تدريجياً من الكبسولة إلى التكوين والتجميع المتقدم.

## Sections
- `01-Encapsulation/` - data validation, getters and setters, smartphone and TV examples
- `02-Inheritance/` - class hierarchy with Computer/Laptop and WeatherData/Forecast
- `03-Polymorphism/` - dynamic binding using Food, Fruit, and Vegetable
- `04-Composition-Aggregation/` - composition and aggregation examples with smartphone components and room furniture

## Learning Goals
- Clarify OOP basics: Encapsulation, Inheritance, Polymorphism, Composition, Aggregation
- Build reusable Java classes with clear validation
- Practice both console input and object relationships

## Best Practices
- Use `private` fields with validated setters
- Prefer meaningful class and method names
- Use `@Override` in subclasses when reusing behavior
- Keep demo classes simple and easy to run

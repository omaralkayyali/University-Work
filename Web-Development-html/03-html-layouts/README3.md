# 🏗️ الدرس الثالث: عناصر وتقنيات تخطيط الصفحات (HTML Layouts)

[cite_start]تعتمد مواقع الويب غالباً على عرض المحتويات في عدة أعمدة (أشبه بتصميم المجلات والجرائد)[cite: 1]. [cite_start]يقدم هذا الدرس شرحاً لكيفية بناء الهيكل العام للصفحة وتقنيات التخطيط الأربعة[cite: 1].

### عناصر التخطيط الدلالية في HTML5
[cite_start]توفر HTML5 وسوم دلالية واضحة (Semantic Elements) تُسهل على محركات البحث والمتصفحات فهم بنية موقعك[cite: 1]:
* [cite_start]`<header>`: يُعرف ترويسة الصفحة أو القسم العلوي[cite: 1].
* [cite_start]`<nav>`: يحتوي على روابط التنقل والقائمة الرئيسية للموقع[cite: 1].
* [cite_start]`<section>`: يعرّف قسماً أو جزءاً معيناً داخل المستند[cite: 1].
* [cite_start]`<article>`: يحتوي على موضوع أو محتوى مستقل بذاته (مثل المقالات)[cite: 1].
* [cite_start]`<aside>`: يعرّف محتوى جانبي (شريط جانبي أو Sidebar)[cite: 1].
* [cite_start]`<footer>`: يعرّف تذييل الصفحة أو الجزء السفلي[cite: 1].
* [cite_start]`<details>`: يعرّف محتوى إضافي يمكن للمستخدم فتحه وإغلاقه عند الطلب[cite: 1].
* [cite_start]`<summary>`: يحدد عنواناً بارزاً لعنصر الـ `<details>`[cite: 1].

### تقنيات تخطيط الصفحات الأربعة في CSS
[cite_start]هناك أربعة طرق أساسية لعمل التخطيطات متعددة الأعمدة، ولكل منها مزايا وعيوب[cite: 1]:

1. **إطارات العمل (CSS Frameworks):** الاعتماد على مكتبات جاهزة لتسريع العمل.
2. [cite_start]**الخاصية العائمة (CSS Float Property):** طريقة كلاسيكية سهلة التعلم تعتمد على خاصيتي `float` و `clear`[cite: 1]. [cite_start]عيبها: ترتبط العناصر العائمة بالتدفق الطبيعي للمستند مما قد يقلل مرونة التصميم[cite: 1].
3. [cite_start]**التخطيط المرن (CSS Flexbox):** تضمن هذه التقنية تصرف العناصر بشكل متوقع ومرن عند تغير أحجام الشاشات وتعدد الأجهزة[cite: 1].
4. [cite_start]**تخطيط الشبكة (CSS Grid Layout):** نظام تخطيط ثنائي الأبعاد يعتمد على الصفوف والأعمدة، مما يسهل تصميم الصفحات المعقدة دون الحاجة لاستخدام الفلوت أو التموضع المطلق[cite: 1].

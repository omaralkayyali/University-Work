# 🎨 الدرس الأول: مقدمة في CSS وبنيتها البرمجية

### ما هي لغة CSS؟
* [cite_start]**CSS** تعني **Cascading Style Sheets**.
* [cite_start]هي لغة تنسيق تُستخدم لوصف طريقة عرض المستندات المكتوبة بلغة HTML أو XML.
* [cite_start]**فائدتها:** تتيح لك التحكم في الألوان، الخطوط، الهوامش، والتخطيطات لتتناسب مع مختلف الشاشات والأجهزة. تكمن ميزتها الكبرى في توفير الوقت؛ [cite_start]حيث يمكنك تغيير مظهر مئات الصفحات عبر تعديل ملف CSS واحد فقط.

---

### البنية القواعدية لـ CSS (Syntax)
[cite_start]تتكون قاعدة التنسيق (CSS Rule-set) من جزأين رئيسيين: **المُحدد (Selector)** و **كتلة الإعلان (Declaration Block)**:

```css
selector {
  property: value;
}
